# DesafioDotNet
Desenvolver uma API RESTful que simule uma gestão de tarefas (ToDo) exposto com Swagger

## Utilizados 
  * Visual Studio 2019
  * Sqlite3 (v.3.51.2 x64):
    [https://sqlite.org](https://sqlite.org/download.html)
  * SqliteBrowser (v.3.13.1 x64):
    [https://sqlitebrowser.org](https://sqlitebrowser.org/dl/)
    
## Executar
  * Abrir o arquivo de solução /ToDo/ToDo.sln
  * Definir como inicial o projeto de Apresentacao
  * Executr no IIS Express
  * A tela de swagger aparecerá
